# PADI Open Water Practice Exam

A self-contained, single-file web app for practicing **PADI Open Water Diver** theory. Open it in any browser — no install, no build step, no internet required.

## Features

- **Randomized 50-question exam** drawn from a pool of **150 questions** — every attempt is a fresh, shuffled set, and the answer choices are reshuffled too, so no two runs are alike.
- **One question at a time** with clean multiple choice.
- **Forward / Previous navigation** plus a **jump-to-question grid** (1–50) showing which questions are answered, unanswered, or current — so you can skip back multiple questions at once.
- **Change any answer** freely before finishing.
- **Instant scoring** at the real **75% pass mark**, with a score ring, PASS/FAIL verdict, and correct / incorrect / skipped counts.
- **Full answer review** — every question with your choice vs. the correct answer highlighted.
- **Progress autosaves** (localStorage), so you can close and resume.
- **Fully responsive** for mobile, tablet, and desktop, with keyboard shortcuts (←/→ to move, A–D or 1–4 to answer).

## Usage

Just open [`index.html`](index.html) in any modern browser (double-click it, or drag it into a browser tab). That's the entire app — a single HTML file with no dependencies. It works completely offline.

## Question pool

- **50 questions** from a PADI Open Water practice exam (with the provided answer key).
- **100 additional questions** authored to cover the Open Water curriculum: dive physics (pressure, Boyle's Law, buoyancy), physiology (DCS, narcosis, equalization), equipment, skills & procedures, dive planning (NDLs, surface intervals, flying after diving), environment, safety, and continuing education.
